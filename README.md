# rhino

A lightweight Android library that integrates Rhino JavaScript engine with Kotlin coroutines.

## Installation

<details>
<summary><b>Via Jar</b></summary></details>
You can download a jar from this link:

[GitHub's releases page](https://github.com/peyilo/rhino/releases)

<details>
<summary><b>Via Gradle</b></summary></details>
Or use Gradle: 

```kotlin
dependencies {
    implementation("io.github.peyilo:rhino:0.0.1")
}
```
<details> <summary><b>Via Maven</b></summary></details>

Or Maven:

```xml
<dependency>
    <groupId>io.github.peyilo</groupId>
    <artifactId>rhino</artifactId>
    <version>0.0.1</version>
</dependency>
```
