# rhino

A lightweight Java library that integrates Rhino JavaScript engine with Kotlin coroutines.

## Installation

<details>
<summary><b>Via Gradle</b></summary></details>
use Gradle: 

```kotlin
dependencies {
    implementation("io.github.peyilo:rhino:1.0.2")
}
```
<details> <summary><b>Via Maven</b></summary></details>

Or Maven:

```xml
<dependency>
    <groupId>io.github.peyilo</groupId>
    <artifactId>rhino</artifactId>
    <version>1.0.2</version>
</dependency>
```
